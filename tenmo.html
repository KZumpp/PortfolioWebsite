Second capstone project for Merit America, which was a mock peer-to-peer money transfer application allowing users to send, request and approve transfers of "TE Bucks". Frontend is a command-line application communicating with a RESTful API server in the backend. With my cohort partner, utilized JDBC for core functionality, testing API methods in Postman and authoring efficient SQL statements using pgAdmin dummy data. Built both server-side and client-side of app, with a focus on Spring exception handling, as well as maintaining clean DRY code. Our code does include all the optional use cases, as well as exception handling that prevents users from sending money to themselves, going into the negative, sending an invalid amount, or approving a request that would bankrupt the other user.
