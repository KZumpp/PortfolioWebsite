Digital business card built from scratch using React and ES6 to create static web page featuring functional components and CSS styling. Started with Scrimba React Solo Project 1 code, and customized elements to match website theme, as well as importing custom images as props. Utilized flexbox for the majority of the styling, and with setting smaller icons/logos as background images inside the elements. Webpage is responsive, and will continue to update with links to email, github, etc. to maximize functionality. Ran production build and deployed using Netlify, click here to view deployed site.
